## 🚀 Deskripsi Proyek

Proyek ini merupakan pengembangan aplikasi **Manajemen Tugas Kuliah berbasis Mobile** menggunakan **Flutter** yang terintegrasi dengan **REST API Laravel**. Aplikasi dirancang untuk membantu mahasiswa mengelola tugas kuliah secara lebih terorganisir, mulai dari pencatatan tugas, pengelolaan deadline, hingga pemantauan status penyelesaian tugas.

Dengan adanya aplikasi ini, mahasiswa dapat mengurangi risiko lupa terhadap deadline, meningkatkan produktivitas, serta mempermudah pengelolaan aktivitas akademik dalam satu aplikasi.

---

## 🎯 Tujuan

- Membantu mahasiswa mengelola tugas kuliah secara terstruktur.
- Mempermudah pencatatan dan pemantauan deadline tugas.
- Menyediakan aplikasi mobile yang mudah digunakan.
- Mengintegrasikan aplikasi Flutter dengan REST API Laravel.
- Meningkatkan efektivitas dan produktivitas mahasiswa dalam menyelesaikan tugas.

### 📖 Kesimpulan

Aplikasi **Manajemen Tugas Kuliah Berbasis Mobile** berhasil dikembangkan menggunakan **Flutter** dengan menerapkan integrasi **REST API Laravel**. Seluruh fitur utama seperti autentikasi pengguna, pengelolaan tugas (CRUD), penentuan deadline, dan pemantauan status tugas telah berjalan dengan baik.

Aplikasi ini diharapkan dapat membantu mahasiswa dalam mengelola tugas secara lebih efektif, meningkatkan produktivitas, serta mengurangi kemungkinan keterlambatan dalam menyelesaikan tugas. Pengembangan selanjutnya dapat menambahkan fitur notifikasi deadline, kategori tugas, sinkronisasi cloud, serta kolaborasi antar pengguna.
